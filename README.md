# receipt-splitter
A website that allows users to split fairly their bills.

## The idea
Users will manually copy their receipt on the website, select the various items on it and assign them to a specific person. The site will then calculate the due amount for each person.

## Technologies
Node.js, javascript framework (still to decide), Docker, AWS app runner

## Scope of the project
Experiment with Docker, AWS app runner and CI/CD pipelines

## Possible improvements
Allow user to upload a picture of the receipt implementing an image to text recognition.
